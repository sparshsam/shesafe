# Changelog

All notable changes to SheSafe are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-05-29

### Added

- Initial public release
- Interactive safety map using Leaflet
- Pin placement with category (lighting, harassment, traffic, security, general)
- Safety tag selection (safe, mixed, unsafe)
- Time-of-day filtering
- Pin comments and discussion threads
- Upvote system for community feedback
- Flagging for moderation
- Admin moderation panel
- Supabase Auth with SSR (HTTP-only cookies)
- Guest read-only access
- Row-level security on all database tables
